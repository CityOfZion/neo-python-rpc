## Install

`pip install neo-python-rpc`



### Basic Usage

- Get Height of blockchain
    ```
    >>> from neorpc.Client import RPCClient
    >>> client = RPCClient()
    >>> blockchain_height = client.get_height()
    >>> blockchain_height
    769332
    ```
